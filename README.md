# Student Attendance Tracker

## 📌 Overview
This project is a Python-based application designed to help students track their attendance across multiple subjects. It calculates attendance percentage and alerts users if attendance falls below the required level.

## 🚀 Features
- Add subjects
- Mark daily attendance
- Calculate attendance percentage
- Warning for low attendance (<75%)
- Data stored using JSON file

## 🛠️ Technologies Used
- Python
- JSON (for data storage)

## ▶️ How to Run
1. Download the repository
2. Open terminal in project folder
3. Run:
   python main.py
